# Contributing to GyaanPath 🙏

Thank you for wanting to help! Every contribution —
big or small — directly helps a rural student in India.

---

## Ways to Contribute

### 1. 📝 Add Solutions (No coding needed!)
The biggest need right now is **textbook solutions**.
If you are a teacher or know the subject:

- Open `docs/CONTENT_GUIDE.md`
- Follow the format
- Submit solutions via GitHub Issues or Pull Request

### 2. 🐛 Report Bugs
- Open a GitHub Issue
- Mention your Android version and phone model
- Attach a screenshot if possible

### 3. 💻 Write Code
- Fork the repo
- Create a branch: `git checkout -b feature/your-feature`
- Make changes
- Push and open a Pull Request

### 4. 🌐 Translate
We need translations for:
- Hindi (हिंदी)
- Marathi (मराठी)
- Gujarati
- Tamil
- Telugu

Open an Issue with label `translation` to get started.

### 5. 🧪 Test
Install the APK on a cheap Android phone and report
anything that breaks. Especially test on:
- Android 5, 6, 7 (old phones)
- 1 GB RAM phones
- Small screens (5 inch)

---

## Code Style
- Use `flutter analyze` before committing
- Follow existing file structure
- Add comments for complex logic
- Keep widgets small and reusable

---

## Questions?
Open a GitHub Issue with label `question`.
We respond within 48 hours.

---

*Every line of code you write puts knowledge
in the hands of a child who has none. Thank you. 🇮🇳*
